OpenGLDialog
--------------------

Description:
-------------------

An VC++ example that shows you how to use OpenGL
in a Mfc Dialog.

** Update: aug/2004: opengl controls stealing focus - problem
has been solved. Controls can now be inserted in the same dialog
as the GL control without any weird focus bugs.

** Update: 02-14-2002: Problems when using a perspective
projection have been solved.
